## The browser extension for e-mail.love, a secure email forwarding service. 

```
Download -> Unzip

# Chrome Install 
Open: chrome://extensions -> Developer mode(on) -> Load unpacked -> browser-extension(folder)

# Firefox Install
Open: about:debugging#/runtime/this-firefox -> Load Temporary Add-on... -> manifest.json

# Edge Install
Open: edge://extensions ->  Developer mode(on) ->  Load unpacked -> browser-extension(folder)

# Brave Install
Open: brave://extensions ->  Developer mode(on) ->  Load unpacked -> browser-extension(folder)

It's pretty much the same for other browsers.
```
